## 飞轮系统升级记录 (2026-03-21 20:40)

### 新增组件
1. **recipe-flywheel.sh** - 核心飞轮脚本
   - IntentGate 意图分析
   - Task Selector 智能选任务
   - Build Verifier 构建验证
   - Self-Critique 自我评判

2. **recipe-flywheel-daemon.sh** - 守护进程
   - 7x24 持续自我进化
   - 工作时间每小时执行

3. **recipe-smart-task-gen.sh** - 智能任务生成
   - 基于代码分析自动生成任务
   - 热点追踪

4. **recipe-self-critique.sh** - 自我评判
   - 4维度评分 (构建/代码质量/回归/性能)
   - 70分阈值

### 配置更新
- .recipe-flywheel-config.json - 飞轮配置
- .recipe-flywheel-state.json - 状态追踪
- .recipe-flywheel-tasks.md - 任务池

### 自循环机制
1. IntentGate → 分析任务复杂度
2. Task Selector → 选择合适任务
3. Hephaestus Executor → 执行任务
4. Build Verifier → 构建验证
5. Self-Critique → 质量评判
6. Loop → 返回步骤1

### 防止重复机制
- 同一任务24小时内不重复
- 热点任务4小时冷却
- 相似任务6小时冷却
