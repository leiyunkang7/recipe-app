# /add-recipe

交互式添加新食谱。

## 用法

```
/add-recipe
```

## 说明

启动交互式食谱创建流程：
1. AI 会请求你上传食谱截图
2. 自动识别提取食谱信息
3. 展示提取结果供确认或修改
4. 确认后创建到数据库

## 示例

```
用户: /add-recipe

AI: 好的！请上传食谱截图，我会帮你识别提取信息。
    支持上传多张截图（如：封面+食材+步骤）。

用户: [上传截图]

AI: 我已识别并提取以下信息：
    [显示提取结果]
    确认创建？
```

## 支持的截图来源

- 下厨房
- 小红书
- 美食杰
- 豆果美食
- 香哈菜谱
- 其他食谱App/网站截图
