# 📋 PRD 文档 - 食谱详情页优化 V1.0

**项目名称**: 食谱 APP 详情页优化  
**版本**: V1.0  
**日期**: 2026-03-08  
**产品经理**: product_manager

---

## 1. 需求背景

详情页是用户查看食谱的核心页面，当前功能基本完整，但交互体验有优化空间。

## 2. 需求范围

### 2.1 功能范围
- [ ] 步骤分步引导（当前折叠展示 → 分步展示）
- [ ] 食材清单（添加checkbox交互）
- [ ] 营养信息卡片
- [ ] 收藏功能

### 2.2 非功能范围
- [ ] 响应式适配
- [ ] 性能优化

---

## 3. 需求详情

### 3.1 步骤分步引导

**当前**: 全部步骤折叠展示  
**优化后**: 
- 大步骤卡片展示
- 点击展开详情
- 当前步骤高亮

### 3.2 食材清单

**当前**: 纯文本列表  
**优化后**:
- checkbox 可勾选
- 已完成食材划线

### 3.3 营养信息卡片

- 显示卡路里、蛋白质、碳水、脂肪
- 简洁卡片展示

### 3.4 收藏功能

- 心脏图标收藏
- 点击切换状态
